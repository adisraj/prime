Manage Unstructured Data
Unstructured data in Retail calculus is maintained logically as Drops in a container or bucket named Datalake
Datalake
A data lake is a centralized repository that allows you to store all your unstructured data at scale.
Streams
A stream is a channel defined as part of Datalake which controls the in and out flow of elements into the datalake.A stream can be thought of as conveyor belt attached to lake which controls the flow of data into datalake.
Drops
Drop is the actual physical entity you are planning to add to the lake by placing into the streams.
Max File size limit
Maximum size supported for file upload is 60 MB.
