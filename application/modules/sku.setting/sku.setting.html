Type [Automatic/Manual]
If type is automatic then everytime a SKU is created a unique number is generated by system and assigned to SKU. If type is manual then SKU number need to be assigned by user during SKU creation.
Algorithm [EAN-13]
The International Article Number (also known as European Article Number or EAN) is a standard describing a barcode symbology and numbering system used in global trade to identify a specific retail product type, in a specific packaging configuration, from a specific manufacturer.
The 13-digit EAN-13 number consists of four components:
  • GS1 Prefix - 3 digits
  • Manufacturer Code - variable length
  • Product Code - variable length
  • Check digit
