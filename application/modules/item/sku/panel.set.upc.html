UPC (technically refers to UPC-A) consists of 12 numeric digits, that are uniquely assigned to each trade item. Along with the related EAN barcode.

Generally, in the UPC-A system, the check digit is calculated as follows:

  1. Sum the digits at odd-numbered positions (first, third, fifth,..., eleventh).
  2. Multiply the result by 3.
  3. Add the digit sum at even-numbered positions (second, fourth, sixth,..., tenth) to the result.
  4. Find the result modulo 10 (i.e. the remainder, when divided by 10).
  5. If the result is not 0, subtract the result from 10.
